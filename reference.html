Introduction

This is the DSL reference for the gradle-blender project. Start reading by finding the right type for the task you are using.

Task types

The gradle-blender project is made of 3 Gradle task types:

Task Description
RenderImage Render a Blender scene to an image file.
RenderAnimation Render a Blender animation to images files.
FbxConv Convert FBX files to G3DJ or G3DB formats.
